KrishiSense – Smart IoT Farm Monitoring Dashboard 🌱
📌 Project Overview

KrishiSense is a Smart IoT Farm Monitoring Dashboard designed to monitor agricultural field conditions in real time. The system displays important farming parameters like soil moisture, temperature, humidity, rainfall, wind speed, soil pH, and EC levels through an interactive dashboard.

This project helps farmers make better decisions for crop management using IoT-based monitoring and data visualization.

🚀 Features
🌡️ Real-Time Temperature Monitoring
💧 Soil Moisture Detection
☁️ Humidity Tracking
🌧️ Rainfall Monitoring
🌬️ Wind Speed Monitoring
🧪 Soil pH Analysis
⚡ EC Level Monitoring

📊 Dashboard Modules

Module	Description

Temperature	Displays current field temperature

Humidity	Shows humidity percentage

Soil Moisture	Detects water level in soil

Rainfall	Tracks rainfall data

Wind Speed	Measures air speed

Soil pH	Indicates soil acidity/basicity

EC Level	Displays soil conductivity
